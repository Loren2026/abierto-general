# Decisiones del proyecto

## Regla
Aquí solo se registran decisiones aprobadas por Loren.

---

## 2026-05-03

### Decisión 001
**Tema:** Hub central de coordinación
**Decisión:** El VPS será el hub operativo principal.
**Motivo:** Privacidad, control y menor fricción que GitHub como centro de trabajo diario.

### Decisión 002
**Tema:** Roles
**Decisión:** Loren será director, Claude actuará como arquitecto/ingeniero jefe y OpenClaw como ejecutor/programador.
**Motivo:** Reparto claro de funciones y menos solapamiento.

### Decisión 003
**Tema:** Uso de GitHub
**Decisión:** GitHub se usará para versiones estables, respaldo estructural y publicación, no como centro operativo diario.
**Motivo:** Evitar dependencia de accesos externos y reducir vulnerabilidad operativa.

### Decisión 004
**Tema:** Inicio de Bloque 1
**Decisión:** Comenzar por la estructura base del sistema de coordinación y los archivos de control.
**Motivo:** Necesidad de una base común antes de desarrollar el panel y la app.
