# Roadmap - Escritorio Maestro

## Visión
Crear un sistema estable de coordinación entre Loren, Claude y OpenClaw, y convertir el actual Escritorio en una app real, modular y preparada para crecer.

## Proyectos iniciales del sistema
- GestActas
- Fit Loren
- Escritorio Maestro

## Fase 0 - Coordinación y base operativa
- Crear estructura en VPS
- Crear archivos de control
- Definir flujo de trabajo entre los tres
- Establecer reglas de uso de GitHub, VPS y chat

## Fase 1 - Panel de coordinación MVP
- Diseñar arquitectura inicial del panel
- Modelo multi-proyecto
- Dashboard general de proyectos
- Vista de estado, bloque actual, decisiones e incidencias
- Seguridad privada básica

## Fase 2 - Base técnica del Escritorio Maestro
- Definir stack técnico
- Crear estructura de app
- Navegación principal
- Persistencia local y base de datos

## Fase 3 - Operativa real
- Alta de proyectos nuevos
- Gestión de estados
- Registro de decisiones
- Seguimiento de bloques
- Historial de cambios

## Fase 4 - Productividad y automatización
- Acciones rápidas
- Exportaciones
- Resúmenes automáticos
- Plantillas de proyecto

## Fase 5 - Integración de IA
- Integración segura con Claude
- Asistencia contextual
- Propuestas de mejora por proyecto
- Soporte a planificación y redacción

## Fase 6 - Consolidación
- Pulido de UX
- Backups
- Versiones estables
- Mantenimiento continuo
