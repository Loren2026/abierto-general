# Propuesta de desarrollo, Bloque 1, Fit Loren

## Objetivo
Preparar la estructura base de la aplicación Flutter `fit_loren` desde cero, sin reutilizar código de prototipos HTML anteriores, dejando un proyecto limpio, ordenado y listo para seguir creciendo por bloques.

## Alcance del Bloque 1
Se propone desarrollar los siguientes elementos:

### 1. Creación del proyecto base Flutter
- Nombre del proyecto: `fit_loren`
- Plataformas objetivo: Android e iOS
- Proyecto inicial limpio, generado desde cero
- Sin incorporar código previo de prototipos HTML ni adaptaciones heredadas

### 2. Configuración visual inicial
Se configurará un tema base coherente con la identidad visual indicada:
- Fondo oscuro: `#1a1a1a`
- Color de acento naranja: `#E8732A`
- Texto principal en blanco

Esta configuración servirá como base común para las siguientes pantallas y componentes.

### 3. Navegación principal
Se implementará una navegación inferior con 5 pestañas:
- Inicio
- Ejercicios
- Entrenar
- Recuperar
- Perfil

Cada pestaña tendrá su pantalla correspondiente, inicialmente vacía pero completamente navegable, para dejar establecida la estructura funcional de la app desde el primer bloque.

### 4. Organización del proyecto
La estructura de carpetas quedará preparada por módulos con esta base:
- `lib/screens`
- `lib/widgets`
- `lib/models`
- `lib/services`

El objetivo es dejar una base clara, mantenible y preparada para el crecimiento posterior del proyecto.

## Resultado esperado
Al finalizar este bloque, Fit Loren quedará con:
- Proyecto Flutter limpio creado desde cero
- Base visual oscura con acento naranja
- Navegación inferior funcional con 5 secciones
- Pantallas iniciales vacías y accesibles
- Estructura de carpetas organizada para futuras ampliaciones

## Observación importante
Esta propuesta solo define el trabajo del Bloque 1.
No se ejecutará ningún desarrollo ni se realizarán cambios en el proyecto hasta recibir autorización expresa de Lorenzo.
